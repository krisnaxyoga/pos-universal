## Screenshot Guidelines

- **Resolution**: Minimum 1920x1080 (Full HD)
- **Format**: PNG or JPG
- **Quality**: High quality, no compression artifacts
- **Content**: Use sample/demo data, no real customer information
- **Browser**: Use Chrome or Firefox in incognito mode (no extensions)
- **Theme**: Ensure consistent theme across all screenshots

## How to Take Screenshots

### Using Browser DevTools

1. Open Chrome DevTools (F12)
2. Click on device toolbar (Ctrl+Shift+M)
3. Set resolution to 1920x1080
4. Take screenshot using DevTools menu

### Using System Tools

- **Windows**: Win + Shift + S
- **macOS**: Cmd + Shift + 4
- **Linux**: gnome-screenshot or flameshot

## Video Demo Requirements

For CodeCanyon submission, also prepare a video demo:

- **Duration**: 2-3 minutes
- **Format**: MP4 (H.264)
- **Resolution**: 1920x1080 (Full HD)
- **Audio**: Background music (royalty-free) + voiceover (optional)

1. **Introduction** (10 sec)

3. **POS Transaction** (40 sec)
